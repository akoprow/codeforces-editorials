1435B A New Technique [CF] (Code) R:1100 constructive algorithms implementation

First find the first column, let’s call it \(C\). Then use it to order rows: row for position \(i\) must have \(row_{i, 1} = C_{i}\). To find \(C\) just take column that contains \(rows_{1, 1}\).

1435C Perform Easily [CF] (Code) R:1900 brute force data structures dp sortings two pointers

Create \((dist, noteIdx)\) array. Sort lexicographically. Now, among all subsequences containing all the notes (i.e. \(\sum_{i \in [l, r]} A[i].noteIdx = {1, .. n}\)) we want to pick the one where cost (i.e. \(A[r].dist - A[l].dist\)) is minimal.

1435D Shurikens [CF] (Code) R:1700 data structures greedy schedules

Picking shuriken \(x\) at a given location means that all the others pending locations must have shurikens \(> x\). This means the last available slot is least constrained so greedily always allocate next shuriken to latest available position (using a stack to keep track of them). Then verify that this is a valid solution by using a heap and checking exercise constraints.
